Spectra Client Installation and Setup Tutorial

1. 下載 Spectra Client

前往 官方下載頁面，取得安裝檔並完成安裝。

2. 啟動 Spectra Client

安裝後先啟動 Spectra Client，並確認連接伺服器與 Riot ID 正確。

3. 開啟《特戰英豪》

請先啟動 Spectra Client，再開啟《特戰英豪》，以避免發生錯誤。

4. 確認開啟順序

請確認 Spectra Client 啟動後，才開啟遊戲，並再次檢查 Riot ID 是否正確。
