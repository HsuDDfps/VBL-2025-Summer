Spectra Client 使用注意事項

Spectra Client 是一款協助主播與賽評即時顯示 VALORANT 戰局資訊的 Overlay 工具。

請務必注意以下幾點以確保順利使用：

  • 僅支援 Windows 平台，請勿在 macOS / Linux 使用。
  • 啟動 Spectra Client 前請先開啟 VALORANT 並完成登入。
  • 如遇無法連線、未顯示資料等狀況，請重新啟動程式並確認網路連線穩定。
  • 請勿重命名主程式檔案，否則可能導致無法更新或錯誤。
  • 程式不會儲存任何帳號密碼資訊，請安心使用。
